Maklumat Sekolah
Google Sheets Sync
Deploy Apps Script sebagai Web App dengan akses "Anyone". Salin URL dan tampal di sini. URL ini juga digunakan untuk jana RPH dengan AI (melalui proxy).
💡 Tip: Sync (Ctrl+S) = simpan data ke server, pantas ~2 saat. Kemas kini Sheets = eksport ke Google Sheets, 5-15 saat. Pulihkan dari Sheets = jika data server salah, baca semula dari tab Google Sheets.
Jana RPH dengan AI
Untuk jana RPH, Apps Script perlu dikonfigurasi dengan Anthropic API Key.
Di Apps Script: Project Settings → Script Properties → Tambah property:
ANTHROPIC_API_KEY = kunci API anda dari console.anthropic.com
Jangan masukkan API key terus dalam app. Simpan dalam Script Properties untuk keselamatan.
